# Claude Usage Tracker

[日本語版 README はこちら](README_ja.md)

A Windows desktop app that visualizes Claude Code token consumption and session usage in real-time.

![Python](https://img.shields.io/badge/Python-3.10+-blue)
![Platform](https://img.shields.io/badge/Platform-Windows-0078D6)
![License](https://img.shields.io/badge/License-MIT-green)

## Features

- **Zero Setup** — Just launch if you're a Claude Code user (OAuth auto-auth)
- **Real-time Monitoring** — Session remaining / weekly usage on system tray
- **Token Analysis** — Cost breakdown by project and action
- **10-segment Gauge** — Task Manager-style tray icon shows remaining capacity at a glance
- **English / Japanese** — Full i18n support

## Requirements

- **Windows 10/11**
- **Claude Code** installed and `claude login` completed
- Python 3.10+ (when running from source)

## Install

### Installer (Recommended)

Download `ClaudeUsageTracker_Setup_vX.X.X.exe` from [Releases](../../releases) and run.
No admin rights required (installs to `%LOCALAPPDATA%\ClaudeUsageTracker`).

### From Source

```bash
git clone https://github.com/Rurimpa/claude-usage-tracker.git
cd claude-usage-tracker
pip install -r requirements.txt
python main.py
```

## How It Works

| Source | Method | Data |
|--------|--------|------|
| Claude Code local logs | Auto-scan `~/.claude/projects/**/*.jsonl` | Per-project token consumption, action history |
| Anthropic Usage API | OAuth token from `~/.claude/.credentials.json` | Session remaining, weekly usage, extra usage |

### Authentication

No browser cookies or API keys needed.
Automatically reads the OAuth token generated by Claude Code login (`~/.claude/.credentials.json`).

## Feature List

### Dashboard
- Period summary (input/output tokens, cost)
- Time-series token consumption graph (hourly/daily)
- Model distribution pie chart
- Recent 50 messages list
- Period selector: presets (Today/1 Week/1 Month/All Time) + calendar input

### Analysis
- Cost bar chart by project (external projects separated)
- Cost bar chart by action (Read/Write/Bash etc.)
- Switchable via dropdown

### Activity Log
- All Claude Code actions in expandable tree view
- Tool name, target file, tokens, cost
- Sortable by clicking headers

### Tray Icon
- Task Manager-style 10-segment vertical bar gauge
- Color changes with remaining capacity (blue → yellow → red → blink)
- Auto-switches to extra usage gauge when session is exhausted (green)
- Left-click for popup, hover for tooltip

### Settings
- Scan interval (10s to 5min)
- Usage API polling interval
- OAuth auth status display
- Language selection (English / Japanese)
- CSV export

## Notes

- The Usage API endpoint (`api.anthropic.com/api/oauth/usage`) is an undocumented Anthropic API. It may change or be discontinued without notice
- Local log analysis works independently of the API
- Displayed costs are estimates and may differ from actual billing
